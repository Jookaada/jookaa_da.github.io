Programming in C++ Book Project

This project might tread on the "illegal" part of projects. It consists in the preservation of the Pimenta Rodrigues, Pedro Pereira and Manuela Sousa's 1998 book "Programação em C++ - Conceitos Básicos e Algoritmos", one of two volumes which explains in full detail the many concepts of the C++ Programming Language and that serves as a great book as an introduction to programming, since it introduces, in a single book, a LOT of concepts that college grade courses take years to teach, like dynamic memory management, pointers and other great stuff.
